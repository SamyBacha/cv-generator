Guide d'utilisation
✏ Édition directe dans le viewer
Cliquez sur n'importe quel texte (nom, poste, compétences…) pour l'éditer directement. Une mini-barre de mise en forme apparaît lors de la sélection.
📋 Panneau Édition
Le mode Édition affiche un formulaire structuré : ajout / suppression d'entrées, missions, compétences, langues, etc.
Sections
Le menu Sections ▾ permet de masquer ou afficher chaque section du CV (formation, hobbies, projets perso…). Les sections masquées n'apparaissent pas à l'impression.
🌐 Langue FR / EN
Bascule les libellés de sections (À PROPOS ↔ ABOUT, COMPÉTENCES ↔ SKILLS, etc.) entre français et anglais.
🖼 Logo de mission
En mode Visualisation, cliquez sur la zone logo d'une mission pour uploader une image, coller du code SVG ou saisir une URL. En mode Édition les mêmes options sont disponibles dans le formulaire.
💾 Sauvegarder ▾
Menu déroulant avec deux actions :
  • Sauvegarder — enregistre le CV dans le localStorage de ce navigateur. Les données sont restaurées automatiquement à la prochaine ouverture.
  • Supprimer Sauvegarde — supprime uniquement les données de ce CV enregistrées dans ce navigateur (les autres sites ne sont pas affectés). Le CV par défaut est rechargé.
JSON ▾
Menu déroulant avec deux actions :
  • ↑ Importer JSON — charge un fichier .json exporté précédemment pour restaurer un CV.
  • ↓ Exporter JSON — télécharge le CV courant en fichier .json pour le sauvegarder ou le partager.
+ Nouveau CV vierge
Charge un template vide (lorem ipsum) pour repartir de zéro. Les données actuelles non sauvegardées seront perdues.
🖨 Imprimer / PDF
Ouvre la boîte d'impression du navigateur. Pensez à cocher « Graphiques d'arrière-plan » pour que les couleurs et dégradés s'impriment correctement.
